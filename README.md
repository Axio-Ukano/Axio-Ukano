## What I Work On

### Minu's Garden
> Cozy study companion desktop app. Turn study sessions into a rewarding experience — earn hearts, nurture a virtual garden, and stay motivated with pixel-art vibes.

**Stack:** Tauri 2 · React 19 · TypeScript 5.8 · Zustand 5 · Vite 7 · SQLite · Howler 2 · Vitest 4

[![Minu's Garden](https://img.shields.io/badge/Minu's%20Garden-v0.6.0-FFC131?style=for-the-badge&logo=tauri&logoColor=black)](https://github.com/Axio-Ukano/minus-garden)

### FoxTrack Automation
> End-to-end test automation framework for FoxTrack built with Selenium + TypeScript. Implements Page Object Model (POM), singleton WebDriver, ACK verification via Gmail IMAP & OpenPhone API, and full CI/CD integration via GitHub Actions + Jenkins.

**Stack:** Selenium WebDriver · TypeScript · Node.js · POM · GitHub Actions · Jenkins · Xvfb · ESLint · Prettier
